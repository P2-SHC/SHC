## 1. 서비스 개요

시니어를 위한 건강 정보 제공, 건강 상품 판매, AI 맞춤 추천, 건강 알리미를 통합한 플랫폼.

| 항목 | 내용 |
|------|------|
| 대상 사용자 | 시니어 (건강 관심 고령층) |
| 주요 가치 | 건강 정보 + 상품 + AI 추천 + 건강 알리미를 한 곳에서 |
| 라우팅 방식 | State 기반 SPA (URL 변경 없음) |
| 인증 방식 | localStorage 기반 (SHA-256 비밀번호 해싱) |

### 2.3 건강 알리미 팝업 (AlarmNotifier)

앱 전역에 항상 렌더링되어 알람 시간이 되면 모달 팝업을 표시한다.

| 기능 | 설명 |
|------|------|
| 30초 간격 체크 | 현재 시간과 설정된 알람 시간 비교 |
| 알람 팝업 | 약 복용(💊) / 혈당(🩸) / 혈압(❤️) 알람을 아이콘과 함께 모달로 표시 |
| 복수 알람 큐 | 동시에 발생한 알람은 순서대로 표시, 대기 중 개수 (+N개) 함께 노출 |
| 복용 완료 | 약 알람 확인 시 자동으로 잔여량 1개 차감 |
| 재발방지 | 오늘 이미 발생한 알람은 재발생하지 않음 (localStorage 저장) |
| 로그아웃 시 초기화 | 사용자 변경 시 알람 상태 및 발화 기록 리셋 |

## 3. 페이지별 기능 명세

### 3.1 메인 페이지 (MainPage)

#### 3.1.1 인기글 피처링
- 날짜 근접도 + 조회수 조합 점수 상위 1개 게시글 선정
- 제목, 카테고리, 조회수 표시
- 클릭 시 게시글 상세 이동

#### 3.1.2 최신 건강 정보 목록
- 최신 게시글 5개 (날짜순 정렬)
- 카테고리 배지, 제목, 날짜 표시

#### 3.1.3 날씨 위젯 (WeatherWidget)
- OpenWeatherMap API로 현재 위치 날씨 정보 표시
- 표시 항목: 날씨 아이콘, 현재 기온, 체감 기온, 습도
- **상품 추천 없음** — AI 건강 추천 페이지 이동 카드만 표시

#### 3.1.4 미세먼지 위젯 (AirQualityWidget)
- WAQI API로 현재 위치 미세먼지 정보 표시
- 표시 항목: AQI 수치, PM2.5, PM10, 상태 (좋음/보통/민감군/나쁨/매우나쁨/위험)
- **상품 추천 3개** (미세먼지 1개 + 날씨 2개):
  - 미세먼지 관련 상품 1개: `미세먼지` 키워드 상품 중 랜덤, 없으면 전체 랜덤
  - 날씨 관련 상품 2개: 날씨 아이콘 기반 키워드 상품 중 랜덤, 부족하면 랜덤 보충

  | 날씨 아이콘 코드 | 키워드 |
  |----------------|--------|
  | 01 (맑음) | 자외선 |
  | 09/10/11 (비·천둥) | 비 |
  | 13 (눈) | 눈 |
  | 그 외 | 환절기 |

#### 3.1.5 Zen Mode
- 날씨 아이콘 5회 클릭 시 활성화
- 헤더·위젯 등 UI 요소 숨김, 날씨 배경만 표시

## 5. 추천 알고리즘

### 5.1 메인 페이지 인기글
날짜 근접도 + 조회수 가중치 조합 → 상위 1개 선정

### 5.2 게시글 상세 상품 추천
```
1. 전체 상품에 matchCount = (상품 키워드 ∩ 게시글 키워드) 점수 부여
2. recipe/exercise: 같은 category 상품 우선 + 나머지로 보완 → 상위 3개
3. life: category 무관, matchCount 내림차순 → 상위 3개
```

### 5.3 미세먼지 위젯 상품 추천
```

3. 최종: [미세먼지 1, 날씨 2] = 총 3개
```
